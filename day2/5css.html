CSS 란?

CSS 는 HTML 문서의 스타일을 지정하는데 사용하는 언어이다. CSS는 HTML 요소가 어떻게 표시되어야 하는지 설명한다.

  • CSS는 Cascading Style Sheets의 약자이다.
  • CSS 작성은 style 태그에서 1)style 사용하거나 또는 2)style 태그에 작성한다.
  • style 태그에서는 특정 요소에 적용할 선택자(selector)가 필요하다. 참고로 style 태그는 head 안에 작성한다.
  • CSS는 한번에 여러 웹페이지(html파일)에 적용되게 할 수 있다.
  • CSS를 저장하는 파일 확장자는 .css이다.
  • CSS 프로퍼티는 글꼴 크기, 색상, 배경이미지, 테두리 등등이 있다.모든 태그에 공통적인것 또는 특정태그에만 적용되는 것을 구별해서 사용한다.
